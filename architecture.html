🎨 주요 기능

1️⃣ 스플래시 제스처

  • 일몰 배경 애니메이션
  • 손으로 태양 차단
  • 스와이프 제스처 감지
  • 햅틱 피드백
  • 진입 시각 자동 기록

2️⃣ 선크림 정보 설정

  • 제품 형태 선택
  • SPF 등급 설정
  • UV 지수 환경 설정
  • 땀/수영 여부 입력
  • 3단계 마법사

3️⃣ 메인 타이머

  • 원형 진행 아크
  • 색상 상태 변화
  • 권장 도포량 안내
  • 히스토리 기록
  • 빠른 도포 버튼

4️⃣ 푸시 알림

  • 재도포 20분 전 예고
  • 재도포 시간 주 알림
  • 도포량 안내 포함
  • 도포 후 자동 재스케줄
  • 로컬 알림

└── node_modules/ 의존성

🚀 배포 파이프라인

개발 환경
$ npx expo start
빌드 프로필
├── development: 개발용 (테스트)
├── preview: 미리보기 (베타)
└── production: 스토어 배포 (AAB)
배포 명령어
$ npm run build:production
$ npm run submit:android
자세한 사항은 DEPLOYMENT_GUIDE.md 참고
